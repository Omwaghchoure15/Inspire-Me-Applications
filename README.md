# Inspire Me 

**Inspire Me** is a simple Android application built using **Kotlin** and **Jetpack Compose that fetches and displays motivational quotes from a public API.  
The app demonstrates clean API integration, state management, and modern UI development following best practices.

---

##  Features

- Fetches motivational quotes from a public API  
- Displays quote text and author  
- Navigate **Next**  quotes  
- Clean and minimal UI using Jetpack Compose  

---

##  Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **MVVM Architecture**
- **Retrofit** (API calls)
- **ViewModel** (State management)
- **Gson Converter** (JSON parsing)
